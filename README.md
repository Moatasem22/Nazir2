# لمسات كار - نظام زينة السيارات

نظام محاسبي وإداري متكامل لمحلات تنجيد وزينة السيارات، متوافق مع هيئة الزكاة والضريبة والجمارك السعودية (ZATCA).

![لمسات كار](https://via.placeholder.com/150/1A2A3A/C5A028?text=لمسات+كار)

## المميزات الرئيسية
- **لوحة تحكم شاملة**: تعرض المبيعات، الأرباح، عدد السيارات، التنبيهات.
- **إدارة أوامر العمل**: إنشاء أمر عمل، تحديد الخدمات، حساب التكاليف، تتبع الحالة.
- **إدارة الخدمات**: قائمة الخدمات مع أسعار وتكلفة وهامش ربح.
- **إدارة المخزون**: متابعة المواد، الكميات، تنبيهات عند انخفاض المخزون.
- **إدارة العمال**: تسجيل العمال، الأجور، المشاريع المنفذة.
- **النظام المحاسبي**: دفتر الأستاذ، قيود اليومية، متابعة الذمم.
- **الفوترة الإلكترونية**: فواتير متوافقة مع ZATCA (QR code، ضريبة 15%، تصدير XML).
- **المدفوعات**: دعم نقدي، شبكة، تحويل، آجل، تقسيط.
- **التقارير**: قائمة الدخل، تحليلات الأداء، مقارنات شهرية.
- **واجهة مستخدم عربية 100%** بتصميم أنيق (أسود كحلي + ذهبي) يحاكي تطبيقات الأندرويد.
- **دعم التاريخ الهجري** والصلاحيات المتعددة (مالك، مدير، محاسب، مشرف، كاشير).

## التقنيات المستخدمة
- **HTML5** - هيكل الصفحات
- **CSS3** - التصميم والتنسيق (مرن، متجاوب)
- **JavaScript** - التفاعل والتنقل بين الصفحات
- **Font Awesome 6** - الأيقونات
- **Google Fonts (Cairo, Poppins)** - الخطوط
